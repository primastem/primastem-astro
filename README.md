# PrimaSTEM Website

Official website for PrimaSTEM — screen-free coding and mathematics for ages 4–10.

Built with [Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/), based on the [Foxi](https://github.com/oxygenna-themes/foxi-astro-theme) theme.

## Development

```bash
npm install
npm run dev
```

## Deploy

Hosted on [Netlify](https://netlify.com/). Pushes to `master` trigger automatic deploys.
